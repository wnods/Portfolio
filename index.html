Análise de Dados Meteorológicos em Tatuoca

O projeto foi feito a partir de um Backup de dados Meteorológico na Ilha de Tatuoca para cooperar no processamento sismico de marés. Os dados sismicos são colhidos por um equipamento chamado Geode e devido a sua sensibilidade, o Geode detecta bastante pertubações e colhe muitos ruídos, para a diminuição desses Ruídos fazemos a diminuição deles com a filtragem de Ventos, Direção do Vento, Chuvas e Radiação Solar. Foi utilizado a linguagem Python para ler e filtrar os dados, o Excel para uma visualização universal dos Dados.

Apl. do Metódo de Decomposição
Espectral de Lanczos

Este trabalho foi conduzido com o objetivo de estimar o campo magnetotelúrico (MT) multifrequencial, a técnica é custosa devido esta estimativa ser realizada frequência por frequência. Além disso, o tamanho do problema envolvido é considerável e com matrizes esparsas, o que leva a muitos erros de arredondamento no processo de realização da estimativa. Uma solução possível para esses problemas pode ser encontrada na aplicação de aceleradores de convergência. Dentre os vários métodos disponíveis, o método da deconvolução espectral de Lanczos (SLDM) é uma alternativa.
